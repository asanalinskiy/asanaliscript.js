## Скачивание фреймворка
Сначала зайдите в нужную директорию
Потом введите это:
```cmd
npm install ascript
```
Потом будет временный цикл из символов | / - \, это значит что [Node.js/npm](https://www.npmjs.org/) собирает данные ascript, потом будут какие то сообщения - это нормально, значит фреймворк уже скачивается. Для скачивания необходимо [package.json](https://pnpm.io/package_json) (можно сделать его через команду `npm init -y` и будет файл [package.json](https://pnpm.io/package_json)), потом когда всё скачается - появится package-lock.json и папка [node_modules](https://node-modules.org/) (не удаляйте, оно хранит важные папки и файлы для `asanaliscript.js`
|
## Downloading the framework
First, navigate to the desired directory (or [C:\Users\your-name](file:///C:/Users)
Then enter this:
```cmd
npm install ascript
```
Then you'll see a temporary loop of | / - \ characters, which means [Node.js/npm](https://www.npmjs.org/) is collecting ascript data. Then you'll see some messages—this is normal, meaning the framework is already downloading. To download, you need a [package.json](https://pnpm.io/package_json) (you can create one with the command `npm init -y`, and you'll have a [package.json](https://pnpm.io/package_json) file). Once everything is downloaded, you'll see a package-lock.json and a [node_modules](https://node-modules.org/) folder (don't delete it, it stores important folders and files for asanaliscript.js).
